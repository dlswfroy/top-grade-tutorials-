# **App Name**: TopGrade Academy Manager

## Core Features:

- Collapsible Sidebar Menu: A three-dot menu that expands into a vertical sidebar menu on click and collapses when clicked again, offering access to different sections of the application: Dashboard, Add New Student, Add Teacher, Accounting, Question Generation, and Settings.
- Dynamic Dashboard: A dashboard featuring three dynamic, color-coded cards displaying key information at a glance: total students, attendance data (present/absent), and a class-wise live chart for visual data representation.
- Student Management: Allows admins to add new student with student's name, class (dropdown: 6th to 10th grade), father's name, mobile number, monthly fee, and image (with preview functionality before saving).
- Student List: Display of student list by class with all details and images, alongside edit and delete buttons for managing student data.
- Search and Edit: Search functionality via role and class that allows editing of the student profile when that particular student shows up as a popup after the search in order to change info and edit student profile.
- Teacher Management: Ability to add teachers with details such as name, mobile number, and image, displayed in a list format for easy management.
- AI-Powered Question Paper Generation: Generative AI tool to automate question paper creation based on class, subject, chapter, question type (creative, MCQ), number of questions, time limit, and total marks; enables PDF download for exams.

## Style Guidelines:

- Primary color: A soft sky blue (#87CEEB), evoking trust and reliability for the educational platform.
- Background color: Light gray (#F0F8FF), creating a clean and non-distracting environment.
- Accent color: A muted violet (#B0E0E6), analogous to the primary sky blue, offering a gentle contrast for highlighting important elements.
- Body and headline font: 'PT Sans', a modern sans-serif for clear readability and a touch of warmth.
- Code font: 'Source Code Pro' for the rare occurence of code being displayed.
- Clean, minimalist icons for menu items and interactive elements.
- Clean and simple design to improve user experience.
- Subtle transition animations for menu interactions and loading states to enhance user engagement.
